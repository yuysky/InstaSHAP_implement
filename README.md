# InstaSHAP_implement

InstaSHAP - Initial Implementation

## How to use

1. Download `sparse-interaction-additive-networks.zip` from the `others` folder and unzip it.  
   This is the **only non-standard package** required.  
   ```bash
   # Optional: install if needed
   pip install .
   ```
2. We don’t use the official package due to incorrect imports in some of its files.
